# Techaneh changelog

## 6.0.0 - 2026-07-19

### تجربه مطالعه و رابط
- انتقال نشانگر پیشرفت مطالعه به محدوده دید خواننده، با انیمیشن نرم و درصد زنده
- بازطراحی منظومه با عمق چندلایه، واکنش سه‌بعدی به ماوس و لمس و حالت کم‌مصرف
- حذف فضای خالی اضافه زیر فوتر و اصلاح ارتفاع صفحه در موبایل
- پایدارسازی کارت‌های مقایسه، جست‌وجوی سریع‌تر و نمایش بهتر در عرض‌های کوچک
- بازطراحی پنجره دستیار به‌صورت کنسول دسکتاپ و Bottom Sheet موبایل
- نمایش منابع پاسخ دستیار و وضعیت استفاده از دانش داخلی سایت
- بهبود رابط صوتی، انیمیشن موج، ورودی گفتاری و پخش پاسخ
- هماهنگی لوگو، favicon، آیکن برنامه و Open Graph

### دانش و عامل‌ها
- ساخت خودکار `data/knowledge.json` از مقاله‌ها و داده محصولات
- Retrieval داخلی برای آگاه‌کردن دستیار از محتوای تکانه
- محدودسازی دستیار به فناوری، رایانه، هوش مصنوعی و محصولات دیجیتال
- پاسخ کوتاه و صمیمی همراه با پیام مؤدبانه برای پرسش نامرتبط
- DeepSeek به‌عنوان مدل اصلی مقاله و دستیار
- GitHub Models به‌عنوان پشتیبان مقاله‌ها در GitHub Actions
- زنجیره جست‌وجو: GNews، Tavily، Tavily Keyless و Hacker News
- تولید تصویر فقط برای مطالب مهم با آستانه تحریریه بالا

### محصولات
- افزایش watchlist به ۱۵۸ محصول در ۸ گروه اصلی
- حذف ادعای غیرواقعی «تمام محصولات بازار» و جایگزینی با کاتالوگ پایش‌شونده
- چرخش دوره‌ای محصولات برای بررسی تدریجی کل فهرست
- قانون انتشار مشخصات فقط پس از تأیید دو منبع مستقل

### امنیت و کیفیت
- CORS محدود، Rate Limit، Turnstile اختیاری، محدودیت payload و هدرهای امنیتی
- جداسازی کامل کلید مقاله و کلید دستیار
- کنترل خودکار فایل‌های SEO، دانش داخلی، JavaScript، Python، JSON و Secretهای ناخواسته
- بهینه‌سازی Service Worker و راهبرد Network First برای دانش و صفحات

### محدودیت‌های شفاف
- اتصال زنده API، DNS و دامنه بدون Secretهای مالک و استقرار واقعی قابل آزمایش نهایی نیست.
- صدای فارسی ابری یکپارچه نشده است؛ پخش فارسی از موتور داخلی دستگاه استفاده می‌کند.
- watchlist گسترده است اما پوشش تمام مدل‌های جهان را تضمین نمی‌کند.
