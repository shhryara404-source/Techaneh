Before You Start: A Journey into Preparation for Programming

Before writing your first line of code, you must set up your environment. This article walks you step‑by‑step through installing Python, choosing an IDE, writing your first program, and learning essential tools...

Techaneh Editorial

In the previous article, we got acquainted with Python. Now it's time to get our hands dirty. Starting programming requires simple prerequisites: a computer (Windows, Mac, or Linux), internet, and most importantly, a curious and patient mindset. Basic English knowledge is helpful.

Installing Python is straightforward. Go to python.org, download the latest version. On Windows, make sure to check 'Add Python to PATH'. After installation, verify by typing python --version in the command line.

Choose a good IDE. VS Code (free and powerful) with the Python extension, PyCharm (Community Edition is free), or Jupyter Notebook (for data analysis). Even the simple IDLE is enough to start.

Your first program: Hello, World! In VS Code, create a file with .py extension and write: print("Hello, World!") and run it. Congratulations! You've written your first program.

Key tips: type the code yourself, love errors (they're your teachers), follow a reliable source, practise, learn with others, and build small projects.

Get to know the command line (cd, dir, ls). Learn pip for installing libraries (pip install library_name). Don't fear common mistakes like missing colon (:) or indentation errors.
