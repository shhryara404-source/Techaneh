First Steps: Python Programming Fundamentals

Learn about variables, data types, operators, conditionals, loops, and functions in Python. This article explains the most fundamental programming concepts with simple, practical examples...

Techaneh Editorial

In the previous two articles, we got to know Python and its environment. Now we dive into the heart of programming: variables, data types, operators, conditionals, loops, and functions.

Variables are storage locations for data. Defining a variable in Python is simple: name = "Ali", age = 25. Naming rules: start with a letter or underscore, contain letters and numbers, avoid keywords.

Data types in Python include: integers (int), floats (float), strings (str), booleans (bool), lists (list), tuples (tuple), and dictionaries (dict). Use type() to check the type.

Operators:

  • Arithmetic: + - * / // % **
  • Comparison: == != > < >= <=
  • Logical: and or not

Input/Output: print() for display, input() for user input. f‑strings produce beautiful output.

Conditionals: if, elif, else for decision making.

Loops: for to iterate over collections (with range for a specific count), while to run as long as a condition is true.

Functions are defined with def, can take input (parameters) and return output (return). Default parameters are supported.

A practical project: a simple calculator that takes two numbers and an operator from the user and computes the result. These concepts are the ABCs of Python.
