قدم‌هایِ اول؛ مبانیِ برنامه‌نویسی با پایتون

با متغیرها، انواع داده، عملگرها، شرط‌ها، حلقه‌ها و توابع در پایتون آشنا شوید. این مقاله پایه‌ای‌ترین مفاهیم برنامه‌نویسی را با مثال‌های ساده و کاربردی توضیح می‌دهد...

تحریریه تکانه

در دو مقاله‌ی قبل، با پایتون و محیط آن آشنا شدیم. حالا وارد قلب برنامه‌نویسی می‌شویم: متغیرها، انواع داده، عملگرها، شرط‌ها، حلقه‌ها و توابع.

متغیرها محلی برای ذخیره‌ی داده‌ها هستند. تعریف متغیر در پایتون بسیار ساده است: نام = "علی", سن = 25. قوانین نام‌گذاری: با حرف یا زیرخط شروع شود، شامل حروف و اعداد باشد، از کلمات کلیدی استفاده نشود.

انواع داده در پایتون عبارتند از: اعداد صحیح (int)، اعشاری (float)، رشته‌ها (str)، بولین (bool)، لیست‌ها (list)، تاپل‌ها (tuple)، و دیکشنری‌ها (dict). برای تشخیص نوع از تابع type() استفاده کنید.

عملگرها:

  • ریاضی: + - * / // % **
  • مقایسه‌ای: == != > < >= <=
  • منطقی: and or not

ورودی و خروجی: print() برای نمایش و input() برای دریافت ورودی از کاربر. f‑stringها خروجی‌های زیبا می‌سازند.

ساختارهای شرطی: if, elif, else برای تصمیم‌گیری.

حلقه‌ها: for برای پیمایش مجموعه‌ها (با range برای تکرار تعداد مشخص) و while برای اجرا تا زمانی که شرط برقرار است.

توابع با def تعریف می‌شوند، می‌توانند ورودی بگیرند (پارامتر) و خروجی بدهند (return). پارامترهای پیش‌فرض نیز پشتیبانی می‌شوند.

یک پروژه‌ی عملی: ماشین‌حساب ساده که دو عدد و عملگر را از کاربر می‌گیرد و نتیجه را محاسبه می‌کند. این مفاهیم، الفبای پایتون هستند.
